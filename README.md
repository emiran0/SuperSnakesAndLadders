## Installation

- Install **pygame**

```cmd
pip3 install pygame
```

- Just run the _main.py_ file on your favorite CLI (CMD, Visual Studio Code, etc..)

```cmd
python3 main.py
```

## How To Play

- Enter the amount of players and names.
- Just click Roll as it goes through each player's turns!!!
