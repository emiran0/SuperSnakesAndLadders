# Super Snakes & Ladders

A multiplayer Snakes &amp; Ladders game made on python.

## Contributors

- Daniel Cowen
- Emirhan Kartal
- Irfan Satria
- Yanzhi Bao

## How To Play

- Enter the amount of players
- Just click Roll as it goes through each player's turns!!!

## Installation

- Install pygame (pip3 install pygame)
- Just run the main.py file on your favorite CLI (CMD, Visual Studio Code, etc..)
